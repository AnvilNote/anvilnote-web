# นโยบายความเป็นส่วนตัว

**อัปเดตล่าสุด: กรกฎาคม 2026**\
**มีผลบังคับใช้: กรกฎาคม 2026**

AnvilNote ให้ความสำคัญกับสิทธิความเป็นส่วนตัวและสิทธิในการกำหนดตนเองเกี่ยวกับข้อมูลส่วนบุคคลของผู้ใช้งาน นโยบายความเป็นส่วนตัวฉบับนี้จัดทำขึ้นเพื่ออธิบายว่าเมื่อคุณใช้งาน AnvilNote ข้อมูลที่เกี่ยวข้องจะถูกจัดเก็บ ส่งผ่าน ประมวลผล และใช้อย่างไร รวมถึงลักษณะการประมวลผลข้อมูลที่อาจเกี่ยวข้องกับบริการของบุคคลที่สาม

AnvilNote ยึดหลักการออกแบบแบบ **Local-First** คือให้ความสำคัญกับการจัดเก็บข้อมูลในอุปกรณ์ของผู้ใช้เป็นอันดับแรก เว้นแต่คุณจะเปิดใช้งานฟีเจอร์ที่ต้องเชื่อมต่อกับบริการของบุคคลที่สามด้วยตนเอง เนื้อหาเอกสารของคุณโดยหลักการแล้วจะถูกจัดเก็บไว้เฉพาะในอุปกรณ์ของคุณเอง และจะไม่ถูกส่งไปยังเซิร์ฟเวอร์ที่อยู่ภายใต้การดูแลของผู้ดูแลโครงการ AnvilNote

การใช้งาน AnvilNote ถือว่าคุณได้อ่านและเข้าใจเนื้อหาที่กำหนดไว้ในนโยบายฉบับนี้แล้ว

## ข้อ 1 ขอบเขตการบังคับใช้

นโยบายฉบับนี้ใช้บังคับกับเวอร์ชันเดสก์ท็อปและเวอร์ชันเว็บที่โครงการ AnvilNote จัดทำขึ้นอย่างเป็นทางการ ตลอดจนองค์ประกอบโปรแกรมที่เกี่ยวข้องซึ่งเผยแพร่โดยผู้ดูแลโครงการ AnvilNote

นโยบายฉบับนี้ไม่ใช้บังคับกับบริการหรือกรณีดังต่อไปนี้

1. เวอร์ชันแยก (fork) ของ AnvilNote ที่บุคคลภายนอกดัดแปลง บรรจุใหม่ หรือเผยแพร่ด้วยตนเอง;
2. บริการของบุคคลที่สามที่คุณเชื่อมต่อหรือใช้งานผ่าน AnvilNote;
3. การเก็บรวบรวม การประมวลผล และการใช้ข้อมูลโดย GitHub, OpenAI, ผู้ให้บริการระบบปฏิบัติการ, แพลตฟอร์มดาวน์โหลดซอฟต์แวร์ หรือแพลตฟอร์มบุคคลที่สามอื่นใด;
4. เว็บไซต์ บริการ หรือโปรแกรมที่มิได้อยู่ภายใต้การควบคุมหรือการดูแลของผู้ดูแลโครงการ AnvilNote

บริการของบุคคลที่สามดังกล่าวจะประมวลผลข้อมูลที่เกี่ยวข้องตามข้อกำหนดการใช้งาน นโยบายความเป็นส่วนตัว และกฎหมายที่ใช้บังคับของตนเอง

## ข้อ 2 การจัดเก็บและการจัดการข้อมูลในเครื่อง

AnvilNote ไม่กำหนดให้ผู้ใช้ต้องลงทะเบียนบัญชี และไม่กำหนดให้ผู้ใช้ต้องเข้าสู่ระบบบริการคลาวด์ใด ๆ ที่ AnvilNote จัดทำขึ้น

ข้อมูลดังต่อไปนี้ที่คุณสร้างหรือแก้ไขใน AnvilNote โดยหลักการแล้วจะถูกจัดเก็บไว้ในอุปกรณ์ของคุณเองทั้งสิ้น

* เนื้อหาเอกสาร;
* ฉบับร่าง;
* ประวัติเวอร์ชันของเอกสาร;
* ไฟล์ที่นำเข้าหรือแนบเข้ามา;
* การตั้งค่าที่ผู้ใช้กำหนดเอง;
* เทมเพลต ฟอนต์ และข้อมูลในเครื่องอื่นใดที่จำเป็นต่อการทำงานของฟีเจอร์;
* เนื้อหาอื่นใดที่คุณสร้างหรือบันทึกไว้ผ่าน AnvilNote

ขึ้นอยู่กับเวอร์ชันที่คุณใช้งาน ข้อมูลดังกล่าวอาจถูกจัดเก็บไว้ในไดเรกทอรีข้อมูลแอปพลิเคชันของเวอร์ชันเดสก์ท็อป ตำแหน่งไฟล์ที่คุณกำหนด หรือพื้นที่จัดเก็บข้อมูลในเครื่อง (local storage) ที่เว็บเบราว์เซอร์จัดเตรียมไว้ให้

ผู้ดูแลโครงการ AnvilNote จะไม่ได้รับ อ่าน สำรอง หรือเก็บรักษาข้อมูลดังกล่าวโดยอัตโนมัติจากการที่คุณใช้งาน AnvilNote เว้นแต่คุณจะส่งเอกสารให้แก่ผู้ดูแลโครงการด้วยตนเอง มิฉะนั้นผู้ดูแลโครงการจะไม่สามารถเข้าถึงเนื้อหาเอกสารของคุณได้

คุณสามารถจัดการหรือลบข้อมูลในเครื่องได้ผ่านฟังก์ชันที่ AnvilNote ระบบปฏิบัติการ หรือเบราว์เซอร์จัดเตรียมไว้ให้ เนื่องจากผู้ดูแลโครงการ AnvilNote ไม่ได้ถือสำเนาข้อมูลในเครื่องของคุณไว้ จึงไม่สามารถกู้คืนเอกสารที่ถูกลบ เสียหาย หรือสูญหายแทนคุณได้

## ข้อ 3 การเก็บรวบรวม การประมวลผล และการใช้ข้อมูลส่วนบุคคล

ในกรณีการใช้งานแบบออฟไลน์ตามปกติ AnvilNote จะไม่ส่งเอกสาร ฉบับร่าง ประวัติเวอร์ชัน หรือเนื้อหาอื่นใดของคุณไปยังเซิร์ฟเวอร์ที่อยู่ภายใต้การดูแลของผู้ดูแลโครงการ AnvilNote

นอกจากนี้ AnvilNote จะไม่เก็บรวบรวม ประมวลผล หรือใช้ข้อมูลส่วนบุคคลของคุณโดยมีวัตถุประสงค์ดังต่อไปนี้

* การวิเคราะห์พฤติกรรมผู้ใช้;
* การสร้างโปรไฟล์ผู้ใช้เพื่อการโฆษณาหรือการตลาด;
* การติดตามข้ามเว็บไซต์หรือข้ามบริการ;
* การวิเคราะห์เนื้อหาเอกสาร;
* การขาย ให้เช่า หรือแลกเปลี่ยนข้อมูลผู้ใช้;
* การให้เนื้อหาเอกสารแก่ผู้ลงโฆษณาหรือนายหน้าข้อมูล

การดาวน์โหลด ติดตั้ง หรือเรียกใช้งาน AnvilNote เพียงอย่างเดียว จะไม่ทำให้ผู้ดูแลโครงการ AnvilNote ได้รับเอกสารหรือข้อมูลส่วนตัวอื่นใดในอุปกรณ์ของคุณ

## ข้อ 4 โหมดอัจฉริยะ AI

### 4.1 การเปิดใช้งานฟีเจอร์

โหมดอัจฉริยะ AI เป็นฟีเจอร์ทางเลือก AnvilNote จะส่งข้อมูลบางส่วนไปยัง API ที่ OpenAI จัดทำขึ้นตามการดำเนินการของคุณ ก็ต่อเมื่อคุณตั้งค่าคีย์ API ของ OpenAI ด้วยตนเองและเปิดใช้งานโหมดอัจฉริยะ AI อย่างจงใจเท่านั้น

หากคุณไม่ได้ตั้งค่าคีย์ API หรือไม่ได้ใช้งานโหมดอัจฉริยะ AI AnvilNote จะไม่ส่งเนื้อหาเอกสารของคุณอันเนื่องมาจากฟีเจอร์นี้

### 4.2 ข้อมูลที่อาจถูกส่ง

เมื่อใช้งานโหมดอัจฉริยะ AI AnvilNote อาจส่งข้อมูลดังต่อไปนี้ไปยัง OpenAI โดยตรง ตามฟีเจอร์ที่คุณเลือกใช้

* คำสั่ง คำถาม หรือข้อความพรอมป์ (prompt) ที่คุณป้อนเข้าไป;
* เนื้อหาเอกสารที่คุณเลือกหรือระบุด้วยตนเอง;
* บริบทที่จำเป็นต่อการดำเนินการตามที่คุณร้องขอ;
* เนื้อหาไฟล์ที่คุณแนบหรือระบุด้วยตนเอง ในกรณีที่ฟีเจอร์นั้นรองรับไฟล์แนบ;
* ข้อมูลอื่นใดที่คุณเลือกอย่างชัดแจ้งให้แบบจำลอง AI นำไปประมวลผล

AnvilNote จะไม่ส่งเอกสารทั้งหมดในอุปกรณ์ของคุณไปยัง OpenAI โดยที่คุณไม่ได้ดำเนินการหรือเลือกด้วยตนเอง

### 4.3 คีย์ API

คีย์ API ของ OpenAI ที่คุณตั้งค่าไว้จะถูกจัดเก็บไว้เฉพาะในอุปกรณ์ของคุณเท่านั้น

* ในเวอร์ชันเว็บ โดยหลักการแล้วจะจัดเก็บไว้ในเซสชันของเบราว์เซอร์หรือพื้นที่จัดเก็บข้อมูลในเครื่องอื่น ๆ;
* ในเวอร์ชันเดสก์ท็อป โดยหลักการแล้วจะจัดเก็บผ่านกลไกการจัดเก็บข้อมูลอย่างปลอดภัยที่ระบบปฏิบัติการจัดเตรียมไว้ให้

เมื่อใช้งานโหมดอัจฉริยะ AI คีย์ API จะถูกส่งไปยัง OpenAI โดยตรง เพื่อใช้ในการยืนยันตัวตนและการอนุญาตคำขอ API คีย์ API จะไม่ถูกส่งไปยังเซิร์ฟเวอร์ที่อยู่ภายใต้การดูแลของผู้ดูแลโครงการ AnvilNote และผู้ดูแลโครงการ AnvilNote จะไม่เก็บรักษาคีย์ API ไว้แทนคุณ

คุณมีหน้าที่เก็บรักษาคีย์ API ไว้อย่างเหมาะสม และต้องรับความเสี่ยงด้วยตนเองอันเกิดจากการรั่วไหลของคีย์ การถูกใช้งานโดยไม่ได้รับอนุญาต การเกิดค่าใช้จ่าย API หรือการที่บุคคลภายนอกได้รับสิทธิเข้าถึง

### 4.4 การประมวลผลข้อมูลโดยบุคคลที่สาม

ข้อมูลที่ถูกส่งไปยัง OpenAI ผ่านโหมดอัจฉริยะ AI จะถูกประมวลผลโดย OpenAI ตามข้อกำหนดการใช้งาน ข้อกำหนดการประมวลผลข้อมูล นโยบายความเป็นส่วนตัว และการตั้งค่าที่เกี่ยวข้องซึ่งมีผลบังคับใช้ในขณะนั้น

ผู้ดูแลโครงการ AnvilNote มิใช่ตัวแทนของ OpenAI และไม่สามารถควบคุมระยะเวลาการเก็บรักษาข้อมูล มาตรการรักษาความปลอดภัย สถานที่ประมวลผล วิธีการส่งข้อมูลข้ามพรมแดน หรือการดำเนินการอื่นใดเกี่ยวกับข้อมูลที่ OpenAI นำมาใช้ได้

ก่อนใช้งานโหมดอัจฉริยะ AI คุณควรตรวจสอบด้วยตนเองว่า

1. คุณมีสิทธิที่จะให้เอกสารหรือข้อมูลที่เกี่ยวข้องแก่ OpenAI;
2. ข้อมูลดังกล่าวไม่อยู่ภายใต้ข้อจำกัดของสัญญา หน้าที่รักษาความลับ ความลับทางการค้า จรรยาบรรณวิชาชีพ หรือหน้าที่ทางกฎหมายอื่นใด;
3. คุณได้รับความยินยอมหรือมีฐานทางกฎหมายอื่นที่จำเป็นสำหรับการประมวลผลข้อมูลส่วนบุคคลของผู้อื่นแล้ว;
4. คุณได้อ่านและยอมรับข้อกำหนดการใช้งานและนโยบายความเป็นส่วนตัวที่ใช้บังคับของ OpenAI แล้ว

เว้นแต่คุณจะยืนยันแล้วว่ามีสิทธิโดยชอบด้วยกฎหมายและเข้าใจความเสี่ยงที่เกี่ยวข้องอย่างเพียงพอ มิฉะนั้นไม่แนะนำให้ส่งเนื้อหาดังต่อไปนี้ไปยังบริการ AI ของบุคคลที่สาม

* เลขประจำตัวประชาชน หมายเลขหนังสือเดินทาง บัญชีการเงิน หรือข้อมูลระบุตัวตนที่มีความอ่อนไหวสูงอื่น ๆ;
* ข้อมูลทางการแพทย์ สุขภาพ หรือข้อมูลชีวมิติ;
* ผลงานวิจัยที่ยังไม่เปิดเผยต่อสาธารณะ;
* ความลับทางการค้าหรือข้อมูลทางธุรกิจที่เป็นความลับ;
* เนื้อหาที่ได้รับความคุ้มครองตามหน้าที่รักษาความลับทางวิชาชีพของทนายความ แพทย์ ผู้สอบบัญชี หรือผู้ประกอบวิชาชีพอื่น;
* ข้อมูลส่วนบุคคลที่ได้มาโดยไม่ได้รับความยินยอมจากเจ้าของข้อมูล;
* ข้อมูลที่กฎหมาย สัญญา หรือระเบียบขององค์กรห้ามมิให้ส่งออก

## ข้อ 5 การตรวจสอบอัปเดตเวอร์ชัน

เวอร์ชันเดสก์ท็อปของ AnvilNote อาจเชื่อมต่อกับ Releases API สาธารณะของ GitHub เพื่อตรวจสอบว่ามีเวอร์ชันใหม่ให้ดาวน์โหลดหรือไม่

การตรวจสอบอัปเดตเวอร์ชันจะไม่ส่งข้อมูลดังต่อไปนี้โดยเจตนา

* เนื้อหาเอกสารของคุณ;
* ฉบับร่างหรือประวัติเวอร์ชัน;
* คีย์ API ของ OpenAI;
* ชื่อเอกสาร;
* ข้อความที่คุณป้อนเข้าไป;
* ไฟล์ที่คุณแนบไว้ใน AnvilNote

อย่างไรก็ตาม การเชื่อมต่อเครือข่ายใด ๆ อาจทำให้บริการของบุคคลที่สามที่รับคำขอได้รับข้อมูลการสื่อสารทางเครือข่ายทั่วไป เช่น ที่อยู่ IP เวลาที่ส่งคำขอ ข้อมูลระบุตัวตนของเบราว์เซอร์หรือโปรแกรม ข้อมูลระบบปฏิบัติการ และข้อมูลคำขอ HTTP มาตรฐานอื่น ๆ

ข้อมูลดังกล่าวจะถูกประมวลผลโดย GitHub ตามนโยบายความเป็นส่วนตัวและข้อกำหนดการใช้งานของ GitHub เอง ผู้ดูแลโครงการ AnvilNote จะไม่ได้รับหรือเก็บรักษาเนื้อหาเอกสารของคุณผ่านการตรวจสอบอัปเดตเวอร์ชัน

## ข้อ 6 การวิเคราะห์ การโฆษณา และเทคโนโลยีการติดตาม

AnvilNote ไม่มีเครื่องมือดังต่อไปนี้ติดตั้งมาในตัว

* บริการวิเคราะห์พฤติกรรมการใช้งาน;
* เครื่องมือติดตามโฆษณา;
* เครื่องมือติดตามข้ามเว็บไซต์;
* SDK โฆษณาของบุคคลที่สาม;
* เครื่องมือเก็บข้อมูลทางไกล (telemetry) เพื่อสร้างโปรไฟล์ผู้ใช้;
* เครื่องมือเก็บรวบรวมข้อมูลเพื่อวัตถุประสงค์ทางการขายหรือการตลาด

AnvilNote จะไม่ขาย ให้เช่า แลกเปลี่ยน หรือให้เนื้อหาเอกสารของคุณแก่ผู้ลงโฆษณา นายหน้าข้อมูล หรือบุคคลที่สามอื่นใดไม่ว่าด้วยวิธีใด

หากในอนาคตมีการเพิ่มฟีเจอร์การรายงานข้อผิดพลาด สถิติการใช้งาน หรือการเก็บข้อมูลทางไกลอื่นใด AnvilNote จะปรับปรุงนโยบายฉบับนี้ก่อนเปิดใช้งานฟีเจอร์ดังกล่าว และจะจัดให้มีการแจ้งเตือนและกลไกการเลือกที่เหมาะสมตามลักษณะของฟีเจอร์นั้น

## ข้อ 7 แพลตฟอร์มบุคคลที่สามและลิงก์ภายนอก

AnvilNote อาจมีลิงก์เชื่อมโยงไปยัง GitHub, OpenAI, Typst หรือเว็บไซต์และบริการของบุคคลที่สามอื่น ๆ

เมื่อคุณคลิกลิงก์ภายนอกหรือใช้บริการของบุคคลที่สาม การประมวลผลข้อมูลที่เกี่ยวข้องจะไม่อยู่ภายใต้บังคับของนโยบายฉบับนี้ ผู้ดูแลโครงการ AnvilNote ไม่มีอำนาจควบคุมและไม่รับประกันเนื้อหา ความพร้อมใช้งาน ความปลอดภัย วิธีการประมวลผลข้อมูล หรือมาตรการความเป็นส่วนตัวของบริการบุคคลที่สามดังกล่าว

เมื่อคุณส่ง Issue, Discussion, Pull Request หรือเนื้อหาสาธารณะอื่นใดผ่าน GitHub ชื่อบัญชี GitHub เนื้อหาความคิดเห็น ไฟล์แนบ และข้อมูลอื่น ๆ ที่คุณให้ไว้ อาจแสดงต่อสาธารณะตามการตั้งค่าแพลตฟอร์มของ GitHub โปรดอย่าเปิดเผยคีย์ API ข้อมูลส่วนบุคคล เอกสารลับ หรือเนื้อหาอื่นที่ไม่เหมาะสมต่อการเปิดเผยต่อสาธารณะ ไว้ใน Issue หรือ Discussion ที่เป็นสาธารณะ

## ข้อ 8 ความปลอดภัยของข้อมูล

AnvilNote ใช้การออกแบบแบบ Local-First เพื่อลดความเสี่ยงในการรั่วไหลของข้อมูลอันเกิดจากการจัดเก็บเอกสารรวมศูนย์ไว้บนเซิร์ฟเวอร์ระยะไกล อย่างไรก็ตาม การจัดเก็บข้อมูลในเครื่องมิได้หมายความว่าข้อมูลจะปลอดภัยอย่างสมบูรณ์

ความปลอดภัยของข้อมูลของคุณยังคงอาจได้รับผลกระทบจากปัจจัยดังต่อไปนี้

* อุปกรณ์สูญหายหรือถูกขโมย;
* บัญชีระบบปฏิบัติการถูกเข้าถึงโดยไม่ได้รับอนุญาต;
* มัลแวร์ แรนซัมแวร์ หรือเหตุการณ์ด้านความปลอดภัยไซเบอร์อื่น ๆ;
* ฮาร์ดดิสก์หรืออุปกรณ์จัดเก็บข้อมูลเสียหาย;
* ข้อมูลเบราว์เซอร์ถูกล้าง;
* การตั้งค่าสำรองข้อมูลไม่สมบูรณ์;
* คีย์ API รั่วไหล;
* ผู้ใช้ลบข้อมูลผิดพลาดหรือดำเนินการผิดพลาด;
* ช่องโหว่ด้านความปลอดภัยของแพ็กเกจบุคคลที่สามหรือระบบปฏิบัติการ

ขอแนะนำให้คุณใช้มาตรการรักษาความปลอดภัยที่เหมาะสม ซึ่งรวมถึง

* การใช้รหัสผ่านเข้าสู่ระบบอุปกรณ์และการล็อกหน้าจอ;
* การเปิดใช้งานการเข้ารหัสดิสก์;
* การสำรองข้อมูลเอกสารสำคัญเป็นประจำ;
* การติดตั้ง AnvilNote จากแหล่งที่เชื่อถือได้เท่านั้น;
* การอัปเดตระบบปฏิบัติการและ AnvilNote ให้เป็นเวอร์ชันล่าสุดอยู่เสมอ;
* การหลีกเลี่ยงการแชร์คีย์ API ให้ผู้อื่น;
* การตรวจสอบและเพิกถอนคีย์ API ที่ไม่ได้ใช้งานแล้วเป็นประจำ;
* การหลีกเลี่ยงการจัดการเอกสารลับบนอุปกรณ์สาธารณะหรืออุปกรณ์ที่อยู่ภายใต้การควบคุมของผู้อื่น

## ข้อ 9 การเก็บรักษาและการลบข้อมูล

โดยหลักการแล้ว ผู้ดูแลโครงการ AnvilNote จะไม่เก็บรักษาเอกสารในเครื่องของคุณ จึงไม่สามารถกำหนดระยะเวลาการเก็บรักษาข้อมูลดังกล่าวได้

เอกสาร ฉบับร่าง ประวัติเวอร์ชัน และการตั้งค่าในเครื่องของคุณ จะยังคงถูกเก็บรักษาไว้ตามการดำเนินการของคุณและสภาพแวดล้อมของอุปกรณ์ จนกว่าคุณจะลบด้วยตนเอง ล้างข้อมูลเบราว์เซอร์ ลบข้อมูลแอปพลิเคชัน หรือข้อมูลสูญหายไปเนื่องจากอุปกรณ์เสียหาย การรีเซ็ตระบบ หรือสาเหตุอื่นทำนองเดียวกัน

การถอนการติดตั้ง AnvilNote เพียงอย่างเดียวจะลบเอกสารหรือการตั้งค่าที่เกี่ยวข้องไปพร้อมกันด้วยหรือไม่นั้น อาจแตกต่างกันไปตามระบบปฏิบัติการ วิธีการติดตั้ง และตำแหน่งจัดเก็บข้อมูล ก่อนถอนการติดตั้ง คุณควรตรวจสอบให้แน่ใจว่าได้สำรองเอกสารสำคัญไว้อย่างเหมาะสมแล้ว

สำหรับข้อมูลที่เก็บรักษาโดยบริการของบุคคลที่สาม ระยะเวลาการเก็บรักษาและวิธีการลบข้อมูลให้เป็นไปตามนโยบายความเป็นส่วนตัวและการตั้งค่าบัญชีของบริการบุคคลที่สามนั้น

## ข้อ 10 สิทธิเกี่ยวกับข้อมูลส่วนบุคคล

เนื่องจากโดยหลักการแล้วผู้ดูแลโครงการ AnvilNote ไม่ได้ถือครองเอกสารในเครื่องหรือข้อมูลส่วนบุคคลที่เกี่ยวข้องของคุณไว้ โดยทั่วไปจึงไม่สามารถดำเนินการขอเข้าถึง แก้ไข ระงับการใช้ หรือลบข้อมูลดังกล่าวแทนคุณได้

คุณสามารถจัดการข้อมูลที่เกี่ยวข้องได้โดยตรงผ่านอุปกรณ์ ระบบปฏิบัติการ เบราว์เซอร์ หรือบัญชีบริการของบุคคลที่สามของคุณเอง

หากคุณเห็นว่าผู้ดูแลโครงการ AnvilNote ถือครองข้อมูลส่วนบุคคลของคุณไว้ ไม่ว่าจะโดยการรายงานข้อผิดพลาด อีเมล การโต้ตอบผ่าน GitHub หรือวิธีการอื่นใด คุณมีสิทธิร้องขอตามกฎหมายที่ใช้บังคับ ดังนี้

* ขอเข้าถึงหรือขอสำเนา;
* ขอให้จัดทำสำเนาให้;
* ขอให้เพิ่มเติมหรือแก้ไขให้ถูกต้อง;
* ขอให้ระงับการเก็บรวบรวม ประมวลผล หรือใช้ข้อมูล;
* ขอให้ลบข้อมูล

การใช้สิทธิดังกล่าวข้างต้นยังคงอาจอยู่ภายใต้ข้อจำกัดเกี่ยวกับการยืนยันตัวตน หน้าที่ในการเก็บรักษาข้อมูลตามกฎหมาย การป้องกันการใช้สิทธิโดยมิชอบ และกฎหมายที่ใช้บังคับอื่น ๆ

## ข้อ 11 เด็กและผู้เยาว์

AnvilNote มิได้ถูกออกแบบมาเพื่อวัตถุประสงค์ในการเก็บรวบรวมข้อมูลส่วนบุคคลของเด็กหรือผู้เยาว์

เนื่องจาก AnvilNote ไม่กำหนดให้ต้องลงทะเบียนบัญชี และเอกสารโดยหลักการแล้วจะถูกจัดเก็บไว้ในอุปกรณ์ของผู้ใช้ ผู้ดูแลโครงการ AnvilNote โดยทั่วไปจึงไม่สามารถทราบอายุของผู้ใช้ได้

เมื่อผู้เยาว์ใช้งานโหมดอัจฉริยะ AI, OpenAI API หรือบริการบุคคลที่สามอื่นใด ควรตรวจสอบให้แน่ใจว่าการใช้งานนั้นเป็นไปตามข้อจำกัดด้านอายุของบริการที่เกี่ยวข้อง และในกรณีที่จำเป็นต้องได้รับความยินยอมจากผู้แทนโดยชอบธรรมหรือผู้ปกครองก่อน

## ข้อ 12 การเปลี่ยนแปลงนโยบาย

AnvilNote อาจแก้ไขปรับปรุงนโยบายฉบับนี้อันเนื่องมาจากการปรับเปลี่ยนฟีเจอร์ การเปลี่ยนแปลงทางเทคนิค การเปลี่ยนแปลงของบริการบุคคลที่สาม หรือข้อกำหนดทางกฎหมาย

นโยบายฉบับแก้ไขจะประกาศไว้บนเว็บไซต์ทางการหรือหน้าโครงการของ AnvilNote และจะระบุวันที่อัปเดตล่าสุดไว้ที่ส่วนบนของเอกสาร หากเป็นการเปลี่ยนแปลงที่มีนัยสำคัญ เช่น การเพิ่มการเก็บรวบรวมข้อมูล การส่งข้อมูลทางไกล การวิเคราะห์ติดตาม หรือเรื่องอื่นที่อาจกระทบต่อสิทธิของผู้ใช้อย่างมีนัยสำคัญ จะมีการแจ้งเตือนเพิ่มเติมด้วยวิธีการที่เหมาะสม

เว้นแต่จะระบุไว้เป็นอย่างอื่น นโยบายฉบับแก้ไขจะมีผลบังคับใช้นับแต่วันที่ประกาศ
