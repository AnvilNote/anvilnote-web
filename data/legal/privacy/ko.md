# 개인정보처리방침

**최종 업데이트: 2026년 7월**\
**시행일: 2026년 7월**

AnvilNote는 사용자의 개인정보 및 개인정보 자기결정권을 중요하게 생각합니다. 본 개인정보처리방침은 귀하가 AnvilNote를 이용할 때 관련 데이터가 어떻게 저장·전송·처리·이용되는지, 그리고 제3자 서비스와 관련하여 발생할 수 있는 데이터 처리 상황을 설명하기 위한 것입니다.

AnvilNote는 **로컬 우선(local-first)** 설계 원칙을 채택하고 있습니다. 귀하가 제3자 서비스에 연결이 필요한 기능을 스스로 활성화하지 않는 한, 귀하의 문서 내용은 원칙적으로 귀하 자신의 기기에만 저장되며 AnvilNote 프로젝트 관리자가 운영하는 서버로 전송되지 않습니다.

AnvilNote를 이용함으로써 귀하는 본 방침에 기재된 사항을 읽고 이해하였음을 표명하는 것으로 간주됩니다.

## 제1조(적용범위)

본 방침은 AnvilNote 프로젝트가 공식적으로 제공하는 데스크톱 버전, 웹 버전 및 AnvilNote 프로젝트 관리자가 배포하는 관련 프로그램 구성요소에 적용됩니다.

본 방침은 다음 각 호의 서비스 또는 상황에는 적용되지 않습니다.

1. 제3자가 스스로 수정, 재패키징 또는 배포한 AnvilNote 포크(fork) 버전.
2. 귀하가 AnvilNote를 통해 연결하거나 이용하는 제3자 서비스.
3. GitHub, OpenAI, 운영체제 공급자, 소프트웨어 다운로드 플랫폼 또는 기타 제3자 플랫폼이 수행하는 데이터의 수집, 처리 및 이용.
4. AnvilNote 프로젝트 관리자가 관리·통제하지 않는 웹사이트, 서비스 또는 프로그램.

전항의 제3자 서비스는 각자의 서비스 약관, 개인정보처리방침 및 적용 법령에 따라 관련 데이터를 처리합니다.

## 제2조(로컬 데이터의 저장 및 관리)

AnvilNote는 사용자에게 계정 등록을 요구하지 않으며, AnvilNote가 제공하는 클라우드 서비스에 로그인할 것을 요구하지도 않습니다.

귀하가 AnvilNote에서 작성하거나 편집하는 다음 각 호의 데이터는 원칙적으로 귀하 자신의 기기에 저장됩니다.

* 문서 내용
* 초안
* 문서 버전 기록
* 가져오거나 첨부한 파일
* 사용자 환경설정
* 템플릿, 글꼴 및 기능 실행에 필요한 기타 로컬 데이터
* 귀하가 AnvilNote를 통해 작성하거나 보관하는 기타 콘텐츠

귀하가 사용하는 버전에 따라, 전항의 데이터는 데스크톱 버전의 애플리케이션 데이터 디렉터리, 귀하가 지정한 파일 위치, 또는 웹 브라우저가 제공하는 로컬 저장공간에 저장될 수 있습니다.

AnvilNote 프로젝트 관리자는 귀하가 AnvilNote를 이용한다는 사실만으로 전항의 데이터를 자동으로 취득, 열람, 백업 또는 보관하지 않습니다. 귀하가 스스로 문서를 프로젝트 관리자에게 제공하지 않는 한, 프로젝트 관리자는 귀하의 문서 내용에 접근할 수 없습니다.

귀하는 AnvilNote, 운영체제 또는 브라우저가 제공하는 기능을 통해 로컬 데이터를 관리하거나 삭제할 수 있습니다. AnvilNote 프로젝트 관리자는 귀하의 로컬 데이터 사본을 보유하지 않으므로, 삭제, 손상 또는 분실된 문서를 대신 복구할 수 없습니다.

## 제3조(개인정보의 수집, 처리 및 이용)

일반적인 오프라인 이용 상황에서 AnvilNote는 귀하의 문서, 초안, 버전 기록 또는 기타 콘텐츠를 AnvilNote 프로젝트 관리자가 관리하는 서버로 전송하지 않습니다.

AnvilNote는 또한 다음 각 호의 목적으로 귀하의 개인정보를 능동적으로 수집, 처리 또는 이용하지 않습니다.

* 사용자 행태 분석의 구축
* 광고 또는 마케팅을 위한 사용자 프로파일링
* 웹사이트 또는 서비스 간 교차 추적
* 문서 내용의 분석
* 사용자 데이터의 판매, 대여 또는 교환
* 문서 내용을 광고주 또는 데이터 브로커에게 제공하는 행위

AnvilNote를 단순히 다운로드, 설치 또는 실행하는 것만으로는 AnvilNote 프로젝트 관리자가 귀하의 기기에 있는 문서 또는 기타 개인 데이터를 취득하지 않습니다.

## 제4조(AI 스마트 모드)

### 4.1 기능 활성화

AI 스마트 모드는 선택적 기능입니다. 귀하가 스스로 OpenAI API 키를 설정하고 AI 스마트 모드를 능동적으로 사용하는 경우에 한하여, AnvilNote는 귀하의 조작에 따라 특정 데이터를 OpenAI가 제공하는 API로 전송합니다.

귀하가 API 키를 설정하지 않았거나 AI 스마트 모드를 사용하지 않는 경우, AnvilNote는 해당 기능으로 인해 귀하의 문서 내용을 전송하지 않습니다.

### 4.2 전송될 수 있는 데이터

AI 스마트 모드를 사용할 때, AnvilNote는 귀하가 선택한 기능에 따라 다음 각 호의 데이터를 OpenAI에 직접 전송할 수 있습니다.

* 귀하가 입력한 지시, 질문 또는 프롬프트 텍스트
* 귀하가 능동적으로 선택하거나 지정한 문서 내용
* 귀하가 요청한 작업을 완료하는 데 필요한 맥락(context) 정보
* 해당 기능이 첨부파일을 지원하는 경우, 귀하가 능동적으로 첨부하거나 지정한 파일 내용
* 귀하가 AI 모델의 처리를 위해 명시적으로 제공하기로 선택한 기타 데이터

AnvilNote는 귀하의 조작이나 선택 없이 귀하의 기기에 있는 모든 문서를 임의로 OpenAI에 전송하지 않습니다.

### 4.3 API 키

귀하가 설정한 OpenAI API 키는 오직 귀하의 기기에만 저장됩니다.

* 웹 버전은 원칙적으로 브라우저 세션 또는 기타 로컬 저장공간에 저장됩니다.
* 데스크톱 버전은 원칙적으로 운영체제가 제공하는 보안 저장 메커니즘을 통해 보관됩니다.

AI 스마트 모드를 사용할 때, API 키는 신원 확인 및 API 요청 인증을 위하여 OpenAI로 직접 전송됩니다. API 키는 AnvilNote 프로젝트 관리자가 관리하는 서버로 전송되지 않으며, AnvilNote 프로젝트 관리자가 대신 보관하지도 않습니다.

귀하는 API 키를 적절히 관리하여야 하며, 키 유출, 무단 사용, API 비용 발생 또는 제3자의 접근 권한 취득으로 인하여 발생하는 위험을 스스로 부담합니다.

### 4.4 제3자의 데이터 처리

AI 스마트 모드를 통해 OpenAI로 전송된 데이터는 OpenAI가 그 당시 유효한 서비스 약관, 데이터 처리 조항, 개인정보처리방침 및 관련 설정에 따라 처리합니다.

AnvilNote 프로젝트 관리자는 OpenAI의 대리인이 아니며, OpenAI가 데이터에 대하여 취하는 보관 기간, 보안 조치, 처리 장소, 국외 이전 방식 또는 기타 데이터 처리 행위를 통제할 수 없습니다.

AI 스마트 모드를 사용하기 전, 귀하는 다음 각 호의 사항을 스스로 확인하여야 합니다.

1. 귀하가 관련 문서 또는 데이터를 OpenAI에 제공할 권한을 보유하고 있을 것.
2. 해당 데이터가 계약, 비밀유지의무, 영업비밀, 직업윤리 또는 기타 법적 의무에 의하여 제한받지 않을 것.
3. 귀하가 타인의 개인정보를 처리하는 데 필요한 동의 또는 기타 적법한 근거를 취득하였을 것.
4. 귀하가 OpenAI에 적용되는 서비스 약관 및 개인정보처리방침을 읽고 이에 동의하였을 것.

귀하가 적법한 권한을 보유하고 관련 위험을 충분히 이해하고 있음을 확인한 경우가 아니라면, 다음 각 호의 내용을 제3자 AI 서비스로 전송하는 것은 권장되지 않습니다.

* 주민등록번호, 여권번호, 금융계좌 또는 기타 고도로 민감한 식별 정보
* 의료, 건강 또는 생체인식 데이터
* 아직 공개되지 않은 연구 성과
* 영업비밀 또는 기밀 상업 데이터
* 변호사, 의사, 회계사 또는 기타 전문직의 비밀유지의무에 의해 보호되는 내용
* 당사자의 동의 없이 취득한 개인정보
* 법령, 계약 또는 조직 규정상 전송이 금지된 데이터

## 제5조(버전 업데이트 확인)

AnvilNote 데스크톱 버전은 다운로드 가능한 새 버전이 있는지 확인하기 위해 GitHub가 공개한 Releases API에 연결할 수 있습니다.

버전 업데이트 확인은 다음 각 호의 데이터를 능동적으로 전송하지 않습니다.

* 귀하의 문서 내용
* 초안 또는 버전 기록
* OpenAI API 키
* 문서 제목
* 귀하가 입력한 텍스트
* 귀하가 AnvilNote에 첨부한 파일

다만, 모든 네트워크 연결은 요청을 수신하는 제3자 서비스로 하여금 IP 주소, 요청 시각, 브라우저 또는 프로그램 식별 정보, 운영체제 정보 및 기타 표준 HTTP 요청 데이터와 같은 일반적인 네트워크 통신 데이터를 취득하게 할 수 있습니다.

전항의 데이터는 GitHub가 자신의 개인정보처리방침 및 서비스 약관에 따라 처리합니다. AnvilNote 프로젝트 관리자는 버전 업데이트 확인을 통해 귀하의 문서 내용을 취득하거나 보관하지 않습니다.

## 제6조(분석, 광고 및 추적 기술)

AnvilNote에는 다음 각 호의 도구가 내장되어 있지 않습니다.

* 사용 행태 분석 서비스
* 광고 추적 도구
* 웹사이트 간 추적 도구
* 제3자 광고 SDK
* 사용자 프로파일링을 위한 원격 측정(telemetry) 도구
* 판매 또는 마케팅을 목적으로 하는 데이터 수집 도구

AnvilNote는 귀하의 문서 내용을 광고주, 데이터 브로커 또는 기타 제3자에게 판매, 대여, 교환하거나 그 밖의 방법으로 제공하지 않습니다.

향후 오류 보고, 사용 통계 또는 기타 원격 측정 기능이 추가되는 경우, AnvilNote는 해당 기능을 활성화하기 전에 본 방침을 업데이트하고, 기능의 성격에 따라 적절한 고지 및 선택 메커니즘을 제공합니다.

## 제7조(제3자 플랫폼 및 외부 링크)

AnvilNote는 GitHub, OpenAI, Typst 또는 기타 제3자 웹사이트 및 서비스로 연결되는 링크를 제공할 수 있습니다.

귀하가 외부 링크를 클릭하거나 제3자 서비스를 이용한 이후의 데이터 처리 행위는 본 방침의 적용을 받지 않습니다. AnvilNote 프로젝트 관리자는 제3자 서비스의 콘텐츠, 가용성, 보안, 데이터 처리 방식 또는 개인정보 보호 조치에 대하여 통제하거나 보증할 책임을 지지 않습니다.

귀하가 GitHub를 통해 Issue, Discussion, Pull Request 또는 기타 공개 콘텐츠를 게시하는 경우, 귀하가 제공한 GitHub 계정명, 게시 내용, 첨부파일 및 기타 데이터는 GitHub의 플랫폼 설정에 따라 공개적으로 표시될 수 있습니다. 공개된 Issue 또는 Discussion에는 API 키, 개인정보, 기밀 문서 또는 기타 공개에 부적절한 내용을 게시하지 마시기 바랍니다.

## 제8조(데이터 보안)

AnvilNote는 로컬 우선 설계를 채택하여 문서가 원격 서버에 집중적으로 저장됨으로써 발생하는 데이터 유출 위험을 낮추고 있습니다. 다만 로컬 저장이 데이터의 절대적인 안전을 의미하지는 않습니다.

귀하의 데이터 보안은 다음 각 호의 요인에 의하여 영향을 받을 수 있습니다.

* 기기의 분실 또는 도난
* 운영체제 계정에 대한 무단 접근
* 악성코드, 랜섬웨어 또는 기타 보안 사고
* 하드디스크 또는 저장장치의 손상
* 브라우저 데이터의 삭제
* 불완전한 백업 설정
* API 키 유출
* 사용자의 오삭제 또는 조작 실수
* 제3자 패키지 또는 운영체제의 보안 취약점

다음과 같은 적절한 보안 조치를 취하실 것을 권장합니다.

* 기기 로그인 비밀번호 및 화면 잠금 사용
* 디스크 암호화 활성화
* 중요 문서의 정기적인 백업
* 신뢰할 수 있는 출처에서 배포한 AnvilNote 버전 설치
* 운영체제 및 AnvilNote를 최신 버전으로 유지
* API 키를 타인과 공유하지 않기
* 더 이상 사용하지 않는 API 키의 정기적인 점검 및 폐기
* 공용 기기 또는 타인이 통제하는 기기에서 기밀 문서를 처리하지 않기

## 제9조(데이터 보관 및 삭제)

AnvilNote 프로젝트 관리자는 원칙적으로 귀하의 로컬 문서를 보관하지 않으므로, 해당 데이터의 보관 기간을 결정할 수 없습니다.

귀하의 문서, 초안, 버전 기록 및 로컬 설정은 귀하의 조작 및 기기 환경에 따라 계속 보관되며, 귀하가 직접 삭제하거나, 브라우저 데이터를 지우거나, 애플리케이션 데이터를 제거하거나, 기기 손상이나 시스템 초기화 등의 사유로 소멸할 때까지 유지됩니다.

AnvilNote를 단순히 제거(삭제)하는 것만으로 문서나 관련 설정이 함께 삭제되는지 여부는 운영체제, 설치 방식 및 데이터 저장 위치에 따라 다를 수 있습니다. 제거하기 전에 중요한 문서가 적절히 백업되었는지 먼저 확인하시기 바랍니다.

제3자 서비스가 보관하는 데이터의 보관 기간 및 삭제 방법은 해당 제3자 서비스의 개인정보처리방침 및 계정 설정에 따릅니다.

## 제10조(개인정보에 관한 권리)

AnvilNote 프로젝트 관리자는 원칙적으로 귀하의 로컬 문서 또는 관련 개인정보를 보유하지 않으므로, 통상적으로 이를 대신하여 열람, 정정, 이용 중지 또는 삭제할 수 없습니다.

귀하는 자신의 기기, 운영체제, 브라우저 또는 제3자 서비스 계정을 통해 직접 관련 데이터를 관리할 수 있습니다.

귀하가 오류 보고, 이메일, GitHub 상호작용 또는 기타 방식을 통해 AnvilNote 프로젝트 관리자가 귀하의 개인정보를 보유하고 있다고 판단하는 경우, 귀하는 관련 법령에 따라 다음 각 호의 사항을 청구할 수 있습니다.

* 열람 또는 조회
* 사본의 발급
* 보완 또는 정정
* 수집, 처리 또는 이용의 중지
* 삭제

전항의 권리 행사는 신원 확인, 법정 보관 의무, 권리 남용 방지 및 기타 관련 법령상의 제한을 받을 수 있습니다.

## 제11조(아동 및 미성년자)

AnvilNote는 아동 또는 미성년자의 개인정보를 수집할 목적으로 설계되지 않았습니다.

AnvilNote는 계정 등록을 요구하지 않고 문서가 원칙적으로 사용자의 기기에 저장되므로, AnvilNote 프로젝트 관리자는 통상적으로 사용자의 연령을 알 수 없습니다.

미성년자가 AI 스마트 모드, OpenAI API 또는 기타 제3자 서비스를 이용하는 경우, 해당 이용이 관련 서비스의 연령 제한을 준수하는지 확인하여야 하며, 필요한 경우 법정대리인 또는 보호자의 동의를 받아야 합니다.

## 제12조(방침의 변경)

AnvilNote는 기능 조정, 기술 변경, 제3자 서비스의 변동 또는 법령상 요구에 따라 본 방침을 개정할 수 있습니다.

개정된 방침은 AnvilNote 공식 웹사이트 또는 프로젝트 페이지에 공고되며, 상단에 최신 업데이트 일자를 표시합니다. 데이터 수집의 추가, 원격 전송, 분석 추적 또는 기타 사용자의 권익에 실질적인 영향을 미칠 수 있는 사항이 포함된 중대한 변경의 경우, 합리적인 방법으로 별도의 고지를 제공합니다.

별도의 설명이 없는 한, 개정된 방침은 공고일부터 효력이 발생합니다.
