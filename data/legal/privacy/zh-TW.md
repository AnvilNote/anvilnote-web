# 隱私權政策

**最後更新日期：2026 年 7 月**\
**生效日期：2026 年 7 月**

AnvilNote 重視使用者的隱私權及個人資料自主權。本隱私權政策旨在說明您使用 AnvilNote 時，相關資料如何儲存、傳輸、處理及利用，以及第三方服務可能涉及的資料處理情形。

AnvilNote 採取**本機優先**的設計原則。除非您主動啟用需要連線至第三方服務的功能，否則您的文件內容原則上只會儲存在您自己的裝置中，不會傳送至 AnvilNote 維護者所管理的伺服器。

使用 AnvilNote，即表示您已閱讀並瞭解本政策所載事項。

## § 1 適用範圍

本政策適用於 AnvilNote 專案正式提供的桌面版、網頁版，以及由 AnvilNote 專案維護者發布的相關程式元件。

本政策不適用於下列服務或情形：

1. 由第三人自行修改、重新封裝或發布的 AnvilNote 分支版本；
2. 您透過 AnvilNote 連線或使用的第三方服務；
3. GitHub、OpenAI、作業系統供應商、軟體下載平台或其他第三方平台所進行的資料蒐集、處理及利用；
4. 非由 AnvilNote 專案維護者控制或管理的網站、服務或程式。

前述第三方服務應依其各自的服務條款、隱私權政策及適用法令處理相關資料。

## § 2 本機資料的儲存與管理

AnvilNote 不要求使用者註冊帳號，亦不要求使用者登入 AnvilNote 所提供的雲端服務。

您在 AnvilNote 中建立或編輯的下列資料，原則上均儲存在您自己的裝置中：

* 文件內容；
* 草稿；
* 文件版本紀錄；
* 匯入或附加的檔案；
* 使用者偏好設定；
* 範本、字型及其他為執行功能所必要的本機資料；
* 其他由您透過 AnvilNote 建立或保存的內容。

依您所使用的版本不同，前述資料可能儲存在桌面版的應用程式資料目錄、您指定的檔案位置，或網頁瀏覽器所提供的本機儲存空間。

AnvilNote 專案維護者不會因您使用 AnvilNote，而自動取得、讀取、備份或保存前述資料。除非您自行將文件提供予專案維護者，否則專案維護者無法存取您的文件內容。

您可以透過 AnvilNote、作業系統或瀏覽器所提供的功能管理或刪除本機資料。由於 AnvilNote 專案維護者不持有您的本機資料副本，因此無法代為復原遭刪除、損毀或遺失的文件。

## § 3 個人資料之蒐集、處理及利用

在一般離線使用情形下，AnvilNote 不會將您的文件、草稿、版本紀錄或其他內容傳送至 AnvilNote 專案維護者所管理的伺服器。

AnvilNote 亦不會基於下列目的主動蒐集、處理或利用您的個人資料：

* 建立使用者行為分析；
* 建立廣告或行銷使用者輪廓；
* 進行跨網站或跨服務追蹤；
* 分析文件內容；
* 銷售、出租或交換使用者資料；
* 將文件內容提供予廣告商或資料仲介業者。

單純下載、安裝或執行 AnvilNote，不會使 AnvilNote 專案維護者取得您裝置中的文件或其他私人資料。

## § 4 AI 智慧模式

### 4.1 功能啟用

AI 智慧模式為選用功能。只有在您自行設定 OpenAI API 金鑰，並主動使用 AI 智慧模式時，AnvilNote 才會依您的操作，將特定資料傳送至 OpenAI 所提供的 API。

如您未設定 API 金鑰，或未使用 AI 智慧模式，AnvilNote 不會因該功能而傳送您的文件內容。

### 4.2 可能傳送的資料

使用 AI 智慧模式時，AnvilNote 可能依您所選擇的功能，將下列資料直接傳送至 OpenAI：

* 您輸入的指令、問題或提示文字；
* 您主動選取或指定的文件內容；
* 為完成您所要求之操作而必要的上下文；
* 您主動附加或指定的檔案內容，如該功能支援附件；
* 其他由您明確選擇提供予 AI 模型處理的資料。

AnvilNote 不會在未經您操作或選取的情況下，主動將您裝置中的全部文件傳送至 OpenAI。

### 4.3 API 金鑰

您所設定的 OpenAI API 金鑰僅會儲存在您的裝置中：

* 網頁版原則上儲存在瀏覽器工作階段或其他本機儲存空間；
* 桌面版原則上透過作業系統提供的安全儲存機制保存。

使用 AI 智慧模式時，API 金鑰會直接傳送至 OpenAI，以供身分驗證及 API 請求授權使用。API 金鑰不會傳送至 AnvilNote 專案維護者所管理的伺服器，亦不會由 AnvilNote 專案維護者代為保管。

您應妥善保管 API 金鑰，並自行承擔因金鑰外洩、遭未授權使用、產生 API 費用或第三人取得存取權限所生之風險。

### 4.4 第三方資料處理

透過 AI 智慧模式傳送至 OpenAI 的資料，將由 OpenAI 依其當時有效的服務條款、資料處理條款、隱私權政策及相關設定進行處理。

AnvilNote 專案維護者並非 OpenAI 的代理人，亦無法控制 OpenAI 對資料所採取的保存期限、安全措施、處理地點、跨境傳輸方式或其他資料處理行為。

在使用 AI 智慧模式前，您應自行確認：

1. 您有權將相關文件或資料提供予 OpenAI；
2. 該等資料不受契約、保密義務、營業秘密、職業倫理或其他法律義務限制；
3. 您已取得處理他人個人資料所需的同意或其他合法依據；
4. 您已閱讀並接受 OpenAI 適用的服務條款及隱私權政策。

除非您已確認具有合法權限並充分理解相關風險，否則不建議將下列內容傳送至第三方 AI 服務：

* 身分證號、護照號碼、金融帳戶或其他高度敏感的識別資料；
* 醫療、健康或生物辨識資料；
* 尚未公開的研究成果；
* 營業秘密或機密商業資料；
* 受律師、醫師、會計師或其他專業保密義務保護的內容；
* 未經當事人同意而取得的個人資料；
* 依法令、契約或組織規範禁止傳輸的資料。

## § 5 版本更新檢查

AnvilNote 桌面版可能連線至 GitHub 公開的 Releases API，以確認是否有可供下載的新版本。

版本更新檢查不會主動傳送下列資料：

* 您的文件內容；
* 草稿或版本紀錄；
* OpenAI API 金鑰；
* 文件標題；
* 您輸入的文字；
* 您在 AnvilNote 中附加的檔案。

然而，任何網路連線均可能使接收請求的第三方服務取得一般網路通訊資料，例如 IP 位址、請求時間、瀏覽器或程式識別資訊、作業系統資訊及其他標準 HTTP 請求資料。

前述資料係由 GitHub 依其隱私權政策及服務條款處理。AnvilNote 專案維護者不會透過版本更新檢查取得或保存您的文件內容。

## § 6 分析、廣告及追蹤技術

AnvilNote 不內建下列工具：

* 使用行為分析服務；
* 廣告追蹤工具；
* 跨網站追蹤工具；
* 第三方廣告 SDK；
* 用於建立使用者輪廓的遙測工具；
* 以銷售或行銷為目的的資料蒐集工具。

AnvilNote 不會出售、出租、交換或以其他方式提供您的文件內容予廣告商、資料仲介業者或其他第三人。

如未來新增錯誤回報、使用統計或其他遙測功能，AnvilNote 將於啟用前更新本政策，並依功能性質提供適當的告知及選擇機制。

## § 7 第三方平台及外部連結

AnvilNote 可能提供指向 GitHub、OpenAI、Typst 或其他第三方網站及服務的連結。

您點選外部連結或使用第三方服務後，相關資料處理行為不受本政策規範。AnvilNote 專案維護者對第三方服務的內容、可用性、安全性、資料處理方式或隱私權措施，不負控制或保證責任。

您透過 GitHub 提交 Issue、Discussion、Pull Request 或其他公開內容時，您所提供的 GitHub 帳號名稱、留言內容、附件及其他資料，可能會依 GitHub 的平台設定公開顯示。請勿在公開的 Issue 或 Discussion 中揭露 API 金鑰、個人資料、機密文件或其他不宜公開的內容。

## § 8 資料安全

AnvilNote 採取本機優先設計，以降低文件因集中儲存於遠端伺服器而產生的資料外洩風險。然而，本機儲存不代表資料絕對安全。

您的資料安全仍可能受到下列因素影響：

* 裝置遺失或遭竊；
* 作業系統帳號遭未授權存取；
* 惡意程式、勒索軟體或其他資安事件；
* 硬碟或儲存裝置損毀；
* 瀏覽器資料遭清除；
* 備份設定不完整；
* API 金鑰外洩；
* 使用者誤刪或錯誤操作；
* 第三方套件或作業系統的安全漏洞。

建議您採取適當的安全措施，包括：

* 使用裝置登入密碼及螢幕鎖定；
* 啟用磁碟加密；
* 定期備份重要文件；
* 安裝可信任來源發布的 AnvilNote 版本；
* 維持作業系統及 AnvilNote 為最新版本；
* 避免將 API 金鑰分享予他人；
* 定期檢查及撤銷不再使用的 API 金鑰；
* 避免在公共或受他人控制的裝置上處理機密文件。

## § 9 資料保存及刪除

AnvilNote 專案維護者原則上不會保存您的本機文件，因此無法決定該等資料的保存期間。

您的文件、草稿、版本紀錄及本機設定，將依您的操作及裝置環境持續保存，直至您自行刪除、清除瀏覽器資料、移除應用程式資料，或因裝置損毀、系統重設等原因而消失。

單純解除安裝 AnvilNote 是否會同時刪除文件或相關設定，可能因作業系統、安裝方式及資料儲存位置而有所不同。解除安裝前，您應先確認重要文件已妥善備份。

由第三方服務保存的資料，其保存期間及刪除方式，應依該第三方服務的隱私權政策及帳號設定辦理。

## § 10 個人資料權利

由於 AnvilNote 專案維護者原則上不持有您的本機文件或相關個人資料，因此通常無從代為查詢、更正、停止利用或刪除該等資料。

您可以直接透過自己的裝置、作業系統、瀏覽器或第三方服務帳號，管理相關資料。

如您認為 AnvilNote 專案維護者因錯誤回報、電子郵件、GitHub 互動或其他方式持有您的個人資料，您得依適用法令請求：

* 查詢或閱覽；
* 製給複製本；
* 補充或更正；
* 停止蒐集、處理或利用；
* 刪除。

前述權利之行使，仍可能受到身分確認、法定保存義務、權利濫用防止及其他適用法令的限制。

## § 11 兒童及未成年人

AnvilNote 並非以蒐集兒童或未成年人個人資料為目的所設計。

由於 AnvilNote 不要求註冊帳號，且文件原則上儲存在使用者裝置中，AnvilNote 專案維護者通常無法得知使用者的年齡。

未成年人使用 AI 智慧模式、OpenAI API 或其他第三方服務時，應確認其使用符合相關服務的年齡限制，並於必要時取得法定代理人或監護人的同意。

## § 12 政策變更

AnvilNote 可能因功能調整、技術變更、第三方服務異動或法令要求而修訂本政策。

修訂後的政策將公布於 AnvilNote 官方網站或專案頁面，並於頁首標示最新更新日期。重大變更如涉及新增資料蒐集、遠端傳輸、分析追蹤或其他可能實質影響使用者權益的事項，將以合理方式提供額外告知。

除另有說明外，修訂後的政策自公布之日起生效。

* [AnvilNote GitHub 組織](https://github.com/AnvilNote)
